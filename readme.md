## 🚀 Installation Guide

Getting this vehicle added to your server is incredibly simple:

1. **Download** the resource package.
2. **Extract** the resource folder and ensure it is named properly (e.g., `CBR250RR`).
3. **Move** the folder into your server's `resources` directory (e.g., `resources/[cars]/CBR250RR`).
4. Ensure the `data/` folder contains your meta files (`handling.meta`, `vehicles.meta`, `carcols.meta`).
5. Open your **`server.cfg`** and add the following line:
   ```cfg
   ensure CBR250RR
   ```
6. **Restart** your server or type `refresh` followed by `ensure CBR250RR` in your live server console.
