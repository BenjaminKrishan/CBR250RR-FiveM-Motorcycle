# Honda CBR250RR (Unbranded) - FiveM Resource

![Honda CBR250RR](https://img.shields.io/badge/FiveM-Resource-orange.svg)
![Version](https://img.shields.io/badge/Version-1.0.0-blue.svg)
![Author](https://img.shields.io/badge/Author-Benjamin_Krishan-green.svg)

A high-quality, ultra-realistic Honda CBR250RR motorcycle mod for FiveM. This resource is unbranded and comes with a variety of tuning options and custom liveries to enhance your server's motorcycle collection.

## 🚀 Features

- **Spawn Code**: `cbr250rr`
- **Tuning Options**:
  - 12 Custom Liveries
  - Multiple Exhaust Systems
  - Custom Fairings (A, B, C variants)
  - Different Tank Designs
  - Custom Seats and Mudguards
- **High Quality Models**: Realistic textures and detailed LODs.
- **Custom Handling**: Optimized for a smooth and realistic riding experience.
- **Optimized**: Performance-friendly assets with proper texture compression.

## 🛠️ Installation

1. Download the resource.
2. Extract the `CBR250RR` folder into your FiveM server's `resources` directory.
3. Open your `server.cfg` file.
4. Add `ensure CBR250RR` to the list of resources to be started.
5. Restart your server or start the resource in-game.

## 📁 Project Structure

- `data/`: Contains meta files (`handling.meta`, `vehicles.meta`, `carcols.meta`, etc.).
- `stream/`: Contains all YTD and YFT asset files.
- `fxmanifest.lua`: Configuration for the resource manifest.
